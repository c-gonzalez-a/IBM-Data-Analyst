# Chat GPT - Generative AI for Data Analysis

This directory contains projects and resources that utilize **Chat GPT** and other Generative Artificial Intelligence tools for data analysis tasks.

## Contents

### Data Files

#### Laptop Pricing Dataset
- **laptop_pricing_dataset_converted.csv** - Original dataset converted
- **laptop_pricing_dataset_filled.csv** - Dataset with missing values filled
- **laptop_pricing_dataset_mod1.csv** - First modified version of the dataset
- **laptop_pricing_dataset_normalized.csv** - Normalized dataset for analysis

### Other Resources

- **prompts.txt** - Collection of prompts used with Chat GPT for various analysis tasks
- **heart+disease/** - Subfolder containing data and analysis related to heart disease

## Objective

This module explores how to use Generative AI tools (ChatGPT) for:
- Data processing and cleaning
- Exploratory data analysis
- Code generation for analysis
- Data visualization creation
- Automated documentation
