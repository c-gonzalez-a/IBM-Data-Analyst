# IBM Data Analyst Certificate Program 📊

A comprehensive repository documenting the complete IBM Data Analyst Professional Certificate program, including course materials, labs, and a real-world final project.

## 📚 Repository Structure

### **Final Project** 🎯
- **Location:** [`Final project/`](Final project/)
- **Description:** Complete data analysis project involving web scraping, data cleaning, exploratory analysis, and visualization
- **Key Notebooks:**
  - [`01.Collecting_Jobs_data_Using_API.ipynb`](Final project/notebooks/01.Collecting_Jobs_data_Using_API.ipynb) - Data collection via APIs
  - [`02.Web-Scraping.ipynb`](Final project/notebooks/02.Web-Scraping.ipynb) - Web scraping techniques
  - [`03.Explore-Data-Set.ipynb`](Final project/notebooks/03.Explore-Data-Set.ipynb) - Exploratory Data Analysis
  - [`04.Finding-Duplicates.ipynb`](Final project/notebooks/04.Finding-Duplicates.ipynb) - Data quality assessment
  - [`05.Removing-Duplicates.ipynb`](Final project/notebooks/05.Removing-Duplicates.ipynb) - Data cleaning
  - [`06.Finding-Missing-Values.ipynb`](Final project/notebooks/06.Finding-Missing-Values.ipynb) - Missing value analysis
  - [`07.Imput-Missing-Values.ipynb`](Final project/notebooks/07.Imput-Missing-Values.ipynb) - Imputation strategies
  - [`08.Normalizing-Data.ipynb`](Final project/notebooks/08.Normalizing-Data.ipynb) - Data normalization
  - [`09.DataWrangling.ipynb`](Final project/notebooks/09.DataWrangling.ipynb) - Data transformation and feature engineering

## 🚀 Getting Started

### Prerequisites
```bash
Python 3.8+
pip or conda
```

### Installation
```bash
# Clone the repository
git clone https://github.com/c-gonzalez-a/IBM-Data-Analyst.git
cd IBM-Data-Analyst

# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Running Jupyter Notebooks
```bash
jupyter notebook
```
