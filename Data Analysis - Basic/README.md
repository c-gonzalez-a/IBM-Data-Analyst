# Módulo 1 — Análisis de Transacciones con Tarjeta de Crédito 🧾

## Descripción 📝
Este módulo contiene un conjunto de datos de transacciones con tarjeta de crédito y un cuaderno de Jupyter (`notebook.ipynb`) que guía el análisis paso a paso. El objetivo es practicar limpieza de datos, análisis exploratorio (EDA), visualización y tareas básicas de modelado aplicadas a transacciones reales.

## Archivos incluidos 📁
- `credit_card_transactions.csv` — dataset de transacciones (IP Address, User ID, Account Number, Age, Shipping Address, Transaction Date, Transaction Time, Transaction Value, Product Category, Units Purchased).
- `notebook.ipynb` — cuaderno con el flujo de trabajo reproducible y ejercicios.

## Objetivos del módulo 🎯
- Cargar y explorar el dataset.
- Identificar y manejar valores faltantes y formatos inconsistentes.
- Limpiar columnas con formatos no estandarizados (fechas, moneda, tipografías en categorías).
- Realizar análisis univariado y bivariado; crear visualizaciones relevantes.
- Detectar anomalías y duplicados; preparar datos para modelado.
- Proponer un ejercicio de modelado (por ejemplo, predicción de transacciones de alto valor).
